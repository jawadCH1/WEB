Contribution Painter

  1. Enter your GitHub repository URL as https://github.com/YOUR_USER_NAME/YOUR_REPO_NAME
  2. Select the year you want to use
  3. Start drawing with click & drag on the graph
  4. Klik "Download script →"
  5. Go to the directory in your terminal where you downloaded the script
  6. Run chmod +x github_painter.sh && ./github_painter.sh
